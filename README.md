# cf-bypass

NodeJS script for ddos that can bypass Cloudflare protection.

# Requisites

1. Installing node
2. npm i cloudscraper
3. npm i randomstring

# Run

```sh
node cf.js <url> <time-in-seconds> [<proxy>]
```

# Docker Run

```sh
docker run -it --rm bobjoe9590/cf-bypass <url> <time-in-seconds> [<proxy>]
```
